# Shiny Paws Admin Panel - Quick Start (Secure Version)

## 🚀 Get Started in 5 Minutes

### What You Can Do

✅ Edit website text content (hero title, descriptions, etc.)
✅ Update pricing for all services
✅ Upload and manage gallery images
✅ View and manage customer bookings
✅ All changes automatically push to your live website
✅ **100% SECURE** - Token is never stored

### Step 1: Create GitHub Token (2 minutes)

1. Go to: https://github.com/settings/tokens
2. Click "Generate new token" → "Generate new token (classic)"
3. Name it: `Shiny Paws Admin`
4. Check the box: `repo` (Full control of repositories)
5. Scroll down and click "Generate token"
6. **Copy the token** and save it in a secure place (password manager)

### Step 2: Login to Admin Panel

1. Visit: https://shinypawsla.com/admin-extended.html
2. Password: `santamonica2025`
3. ✅ You're in! No token setup needed.

### Step 3: Make Your First Edit

**Example: Update Hero Title**

1. Click the **✏️ Content** tab
2. Find "Home Page" section
3. Change "Hero Title" text
4. Click **💾 Save Locally**
5. Click **🌐 Update Website**
6. **Paste your GitHub token** when prompted
7. Wait 1-2 minutes
8. Refresh your website to see the change!

### Step 4: Update Pricing

1. Click the **💰 Pricing** tab

3. Click **💾 Save Locally**

2. Select section: "Gallery Images"

## 🔐 Token Security

### Where to Keep Your Token

✅ **Password manager** (1Password, LastPass, Bitwarden)
✅ **Secure note** (encrypted)
❌ **Browser bookmarks** (not secure)
❌ **Email** (not secure)
❌ **Text files** (not secure)
❌ **Shared documents** (not secure)

### If Token is Compromised

1. Go to: https://github.com/settings/tokens
2. Find "Shiny Paws Admin" token
3. Click "Delete"
4. Create a new token
5. Use new token in admin panel

That's it! The old token becomes useless immediately.

## ❓ FAQ

**Q: Why do I need to paste the token each time?**
A: For security! This way the token is never stored on your computer.

**Q: What if I lose my token?**
A: Create a new one at https://github.com/settings/tokens

**Q: Can someone steal my token from the prompt?**
A: No, the prompt is local to your browser. It never leaves your computer.

**Q: How long does the token last?**
A: As long as you want. You can set expiration when creating it (90 days, 1 year, etc.)

**Q: What if I forget to paste the token?**
A: Just click "Update Website" again and paste it.

**Q: Can I use the same token on multiple computers?**
A: Yes! Just save it securely on each computer.

**Q: How do I check what changed?**
A: Go to https://github.com/ssorvex/ShinyPaws and click "Commits"
